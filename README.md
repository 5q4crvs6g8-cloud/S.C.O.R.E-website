# S.C.O.R.E. - Space Company (Fan Project)

This is a **fan-made website** about a fictional space company called **S.C.O.R.E.** (Space Company Of Research & Exploration).  
The site was created for fun and learning web development, inspired by NASA and space exploration.

## Features

- Fully responsive **HTML site** with embedded CSS and JavaScript.  
- **Language switcher**: English, Polish, Ukrainian.  
- **Projects section** with images and descriptions of fictional space missions:
  - Earth Project
  - Moon Mission
  - Mars Exploration
- **Creators section** with cards for the site's creators (without real personal images).  
- Smooth scrolling with "Learn More" button.
